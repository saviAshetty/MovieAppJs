# Movie Search Web Application

## Overview

The Movie Search Web Application is a responsive web application that allows users to search for movies and view detailed information using the OMDb API. The application provides a clean interface with features such as dark mode, search suggestions, favorites, and search history.

## Features

- Search movies by title
- Display movie poster, title, genre, rating, release date, runtime, language, actors, and plot
- Autocomplete search suggestions
- Dark mode with saved user preference
- Favorite movies using Local Storage
- Search history using Local Storage
- Responsive design for desktop and mobile
- Loading indicator while fetching data

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- OMDb API
- Fetch API
- Local Storage

## How to Run

1. Download or clone the repository.

```bash
git clone https://github.com/your-username/movie-search-app.git
```

2. Open the project folder in Visual Studio Code.

3. Install the Live Server extension.

4. Right-click **index.html** and select **Open with Live Server**.
